# AppTodo
Aplicativo de tarefas feito com React Native e armazenando dados no AsyncStorage do Expo

## Features
* React Native
* Async Storage
* Expo 

## Como rodar?
```
npm install
expo start
```
* Baixar o app do expo no celular ou em algum emulador em sua máquina
* Ler o QRcode
* Assim o app já estará funcionando
